# Task Tracker

Небольшой трекер задач, написанный на **Vanilla HTML, CSS и JavaScript** без фреймворков и библиотек.

Это мой первый самостоятельный проект на HTML, CSS и JS. В нём я собрал базовую логику todo-приложения: работа с DOM, обработка событий, фильтрация задач и сохранение данных в браузере.

## Возможности

- Добавление новых задач
- Удаление задач
- Отметка задач как выполненных
- Фильтрация задач: все, активные, выполненные
- Счётчик активных задач
- Сохранение задач в `localStorage`
- Восстановление списка задач после перезагрузки страницы
- Адаптивная вёрстка простого интерфейса

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage

## Структура проекта

```text
.
├── index.html   # Разметка страницы
├── style.css    # Стили интерфейса
└── script.js    # Логика приложения
```

## Как запустить

Проект не требует сборки или установки зависимостей.

Достаточно открыть файл `index.html` в браузере.

## Что я отработал в проекте

- Создание HTML-структуры приложения
- Стилизация формы, кнопок, фильтров и списка задач
- Добавление элементов в DOM через JavaScript
- Работа с `addEventListener`
- Использование `classList` для изменения состояния элементов
- Работа с чекбоксами и состоянием задач
- Сохранение массива задач через `JSON.stringify`
- Загрузка сохранённых задач через `JSON.parse`

Проект завершён как первая самостоятельная версия. В будущем его можно развить: добавить редактирование задач, даты, категории, drag-and-drop или переключение темы.
